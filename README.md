## installation
1.clone the repository
~~~
git clone https://github.com/Efesint/chudsay
~~~
2. install ruby
~~~
sudo apt install ruby
~~~
or
~~~
sudo pacman -S ruby
~~~
or
~~~
sudo emerge -av ruby
~~~
3. enjoy.
~~~
ruby chudsay.rb
~~~
