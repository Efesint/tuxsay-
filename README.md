# tuxsay- tuxsay is a TUI program written in Ruby. It’s like cowsay, but cooler.
just for fun, nevermind.
## installation
1.clone the repository
~~~
https://github.com/Efesint/tuxsay
~~~
2. install ruby
~~~
sudo apt install ruby
~~~
or
~~~
sudo pacman -S ruby
~~~
or
~~~
sudo emerge -av ruby
~~~
3. enjoy.
~~~
ruby tuxsay.rb
~~~
